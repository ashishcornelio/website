# SangamOne Connected Services – Projects

A modern project showcase website built with **HTML, CSS, and JavaScript**, featuring interactive UI components and a dynamic **Three.js beams background**.

## 🌐 Live Demo

https://ashishcornelio.github.io/website/

## ✨ Features

* Animated **Three.js beams background**
* Interactive **3D hover project cards**
* Modern **glassmorphism UI**
* Responsive grid layout
* Smooth hover animations
* Clean and minimal design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES Modules)
* Three.js

## 🚀 Run Locally

Clone the repository:

```bash
git clone https://github.com/ashishcornelio/website.git
cd website
```

Start a local server:

```bash
npx serve
```

Then open the address shown in the terminal (usually `http://localhost:3000`).

⚠️ **Do not use**

```bash
python -m http.server
```

because this project uses **ES module imports**, which may not load correctly with the Python server.
